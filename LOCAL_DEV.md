# Локальная разработка

## Быстрый старт

1. **Установи зависимости:**
   ```bash
   npm install
   ```

2. **Настрой базу данных:**
   ```bash
   npm run db:generate
   npm run db:push
   ```

3. **Запусти проект:**
   ```bash
   npm run dev
   ```

   Или все сразу:
   ```bash
   npm run local:start
   ```

## Настройки

Проект настроен для работы с SQLite базой данных в файле `dev.db` для простоты разработки.

### Переменные окружения

Файл `.env.local` содержит настройки для локальной разработки:
- `DATABASE_URL="file:./dev.db"` - SQLite база данных
- `NODE_ENV=development` - режим разработки
- `NEXT_PUBLIC_APP_URL=http://localhost:3000` - локальный URL (может быть 3000 или 3001 в зависимости от доступности порта)

### Полезные команды

- `npm run dev` - запуск в режиме разработки
- `npm run db:studio` - открыть Prisma Studio для просмотра БД
- `npm run db:seed` - заполнить БД тестовыми данными
- `npm run db:reset` - сбросить БД

## Доступ к приложению

После запуска приложение будет доступно по адресу: http://localhost:3001 (или другой порт, если 3000 занят)

## Troubleshooting

Если возникают проблемы:

1. Удали `node_modules` и `package-lock.json`, затем выполни `npm install`
2. Удали файл `dev.db` и выполни `npm run db:push`
3. Проверь, что все переменные окружения в `.env.local` настроены правильно
